部署博客

1.搭建博客

首先，去GitHub主页下载 hugo（注意Windows用户需要下载_extend版本，否则运行报错）：
https://github.com/gohugoio/hugo/releases

然后配置环境变量：将hugo.exe所在目录添加到系统环境变量Path下。

打开终端验证环境变量：

hugo version

新建一个文件夹，并在其下利用hugo创建博客

hugo new site myblog

2.下载主题

下载主题喜欢的主题，并且按照主题要求安装，修改 config.toml

https://themes.gohugo.io/

启动博客

hugo server -D

根据提示，进入 localhost:1313 查看结果

3.创建文章

输入命令

hugo new post/first.md

这时在 myblog/content/post 目录下有刚创建的文章

打开文件进行编辑，写成你的笔记

随后输入命令：

hugo server -D

就可以看到修改。

4.部署到github上

1.在Github上新建仓库 username.github.io，注意 一定是 username.github.io，username 为你的 Github 用户名

2.将 myblog/public 目录 push 到刚创建仓库的 master 分支

cd public
git init
git add .
git commit -m "message"
git remote add origin https://github.com/wjl-lab/wjl-lab.github.io.git
git push -u origin master

访问网站 wjl-lab.github.io
